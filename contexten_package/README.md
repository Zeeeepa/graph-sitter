# Contexten

An intelligent agent orchestration framework with chat capabilities, GitHub/Linear integrations, and workflow automation.

## Installation

```bash
cd contexten_package
pip install -e .
```

## Usage

```python
from contexten import FlowManager, BaseAgent

# Create and manage workflows
flow_manager = FlowManager()

# Set up agents for various tasks
agent = BaseAgent()
```

## CLI Usage

After installation, you can use the `contexten` command:

```bash
contexten --help
```

## Dashboard

Launch the interactive dashboard:

```bash
contexten-dashboard
```

## Features

- Agentic orchestration
- Chat-agent capabilities
- LangChain integration
- GitHub integration
- Linear integration
- Slack integration
- Workflow automation
- Interactive dashboard
- Flow management
- Task orchestration
